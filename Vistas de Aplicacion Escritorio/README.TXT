Esta seccion corresponde al modelado del front de la aplicacion de escritorio, 
para poder visualizar y correr el codigo se debe contar con un Entorno de Desarrollo Integrado (IDE)
como puede ser Netbeans o Eclipse, tambien debe contar con JDK13 o superior,
para que al momento de ejecutar el codigo se pueda visualizar e interacturar con las vistas desarrolladas.
